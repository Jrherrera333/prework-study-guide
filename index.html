HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element contains information about the author and the creator of the page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the center inside an element.
  • Inline CSS, Internal CSS, and External CSS style sheet are the three ways to style a web page, with External stylesheet being the most common method.
  • A css selector is the first part of a CSS rule, followed by the property and then the value in the declaration block.
  • The class global attribute in CSS is a space-separated list of the case-sensitive classes of the element.

GIT

  • git status: check what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git log --stat -m: show all commit logs with an indication of any paths that moved
  • git pull: Fetch and merge any commits from the tracking remote brach
  • git push [alias] [branch]: transmits local branch commits to the remote repository branch
  • git merge [alias]/[branch]: merge a remote branch into your current branch to bring it up to date
  • git fetch [alias]: fetch down all the branches from the git remote

Jvascript

  • A variable is a named container that allows us to store data in our code.
  • Conntrol flow is the order in which a computer executes code in a script.
